Data Binding

String Interpolation

Transfer Data from TS to HTML

Data Should be in text format

use {{}} to bind data

Hello {{ fname }}

Hi, {{ user.name }} your id is{{ user.id }}

Property Binding

Transfer Data from TS to HTML

Data Should be in text format

use [] to bind data



Event Binding

Transfer Data from HTML to TS

Generally used for call function as per event

use () to bind data





Two Way Data Binding

Combination of property and event binding

Transfer Data from TS to HTML and HTML to TS

Data should be in text format

use [(ngModel)] to bind data

NOTE:

  • Use ngModel to achieve 2 way data binding
  • to use ngModel import FormsModule in add.module.ts
